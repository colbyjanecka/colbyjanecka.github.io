Parallel Computing “Beowulf” Cluster

After disassembling several discarded desktop computers and optimizing the distribution of components, I was able to create 8 separate Linux machines linked together through a LAN network, which allowed me to experiment with basic server management, shared file systems (NFS), and parallel computing via the OpenMPI library.

  • Services: Ubuntu Server, Nginx, MPI
